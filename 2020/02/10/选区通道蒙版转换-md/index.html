发表于 2020-02-10 | 分类于 摄影后期 | | 阅读次数: |

看完 Thomas 大神的视频Photoshop选区、通道、蒙版关系与用法全解析，觉得还是图文的形式比视频更好理解和记忆。开局一张图，箭头表示二者之间可以实现转换。

选区与通道

选区存储为通道

  • 右侧通道面板右下方四个按键里第二个：将选区存储为通道
  • 快捷键：Alt + S + V （存储选区）

通道转化为选区：蚂蚁线显示的是不透明度50%以上的选区（白色全选，黑色全不选）

  • 按住 Ctrl，点击该通道
  • 快捷键：Alt + S + O （载入选区）
  • 右侧通道面板下方四个按键里第一个：将通道作为选区载入
选区与蒙版

选区转化为蒙版

  • 右侧图层面板右下方六个按键里第二个：添加图层蒙版
  • 右侧图层面板右下方六个按键里第三个：添加调整图层，自动生成图层蒙版

选区替换原来的蒙版

  • 选区转化为通道

  • 将原来蒙版选中，点编辑->填充->黑色

  • 通道转化为选区

  • 将白色蒙版选中，点编辑->填充->白色

通道与蒙版

通道与蒙版之间不能转化，必须依赖于选区作中间桥梁

蒙版与蒙版

蒙版复制

  • 从一个图层复制到另一个图层：按住 ALT 键，然后拖动蒙版到另一个图层

蒙版移动

  • 从一个图层移动到另一个图层：直接拖动到另一个图层

蒙版反向

  • 选中蒙版，按 ctrl + i
